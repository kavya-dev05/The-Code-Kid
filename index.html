Guided Projects

Step-by-step projects for every tier. Each one teaches real concepts through hands-on building.

Project 01
🎮 Junior · Beginner
Colour Mixer App

Tap any two primary colour buttons to see what colour they make together. Builds intuition for variables, conditions and user interaction.

VariablesIf-ElseButtons
1Create Red, Blue and Yellow clickable circles
2Use if-else logic to detect which pair is selected
3Display the mixed colour result with a fun animation
4Add a Reset button and a colour name label
~1 hour

Project 02
🎮 Junior · Intermediate
Animal Sound Quiz

A 10-round quiz game that shows a random animal emoji and asks for its sound. Tracks score, shows streaks and celebrates with confetti at 100%.

ArraysRandomScore Tracker
1Store animals and answers in two matching arrays
2Pick a random animal each round, show 4 choices
3Check answer, update score and show feedback
4End screen showing final score and play-again button
~1.5 hours
🧮
Project 03
🎮 Junior · Advanced
Times Table Trainer

A maths trainer that fires random multiplication questions, races against a 10-second timer, tracks streaks and unlocks a "Speed Star" badge at 10 correct in a row.

FunctionsTimersLoops
1Generate random multiplication questions (1-12 tables)
2Add a countdown timer that resets each question
3Track streak and score — show progress bar
4Award Speed Star badge on 10-streak + leaderboard
~2 hours

Project 01
🔭 Explorer · Beginner
Weather Dashboard

Build a temperature converter and weather card that switches icons based on conditions, stores the last 5 readings in an array and charts them as a simple graph.

FunctionsArraysDOM API
1Write C to F and F to C conversion functions
2Map temperature ranges to weather icons dynamically
3Push each reading to an array (max 5, oldest drops off)
4Render a mini bar chart of the 5 readings using CSS
~2 hours

Project 02
🔭 Explorer · Intermediate
Smart To-Do List

A priority-based task manager with add, complete and delete. Tasks are stored as objects with priority levels, sorted automatically and filtered by status.

ObjectsSort/FilterEvents
1Model each task as an object with title, priority, done
2Sort tasks by priority (High / Medium / Low)
3Filter view: All | Active | Completed tabs
4Add due-date support and overdue highlighting
~2.5 hours
🎯
Project 03
🔭 Explorer · Advanced
Number Guessing Game

A smart guessing game with difficulty levels, Higher/Lower hints, guess counter and a replay screen that reveals the optimal binary search strategy used.

Binary SearchRecursionLogic
1Generate a secret number in a range set by difficulty
2Give "Higher" / "Lower" feedback after each guess
3Count guesses and compare to binary search minimum
4Show a replay of the optimal binary search path
~3 hours
Project 01
Pro · Intermediate
Algorithm Visualiser

AlgorithmsSVG/CanvasBig-O

2Implement both search algorithms with step callbacks
3Plot live comparison count on an SVG complexity graph

Student Grade Analyser

Map/Filter/ReduceSVG ChartsOOP

4Generate a plain-text report card and copy to clipboard
