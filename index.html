Project 01
🎮 Junior · Class 1–3
My Colour Mixer

Build a fun colour mixing app! Click red, blue and yellow circles to mix them and see what new colour appears. Learn about variables and if-statements.

Variables If-Else Colours
1Create 3 colour buttons (Red, Blue, Yellow)
2Use if-else to detect which two are clicked
3Show the mixed colour result in a big circle
🐾
Project 02
🎮 Junior · Class 3–4
Animal Quiz Game

Make a quiz that shows an animal emoji and asks "What sound does this animal make?" Use lists and random numbers to pick animals each round.

Lists/Arrays Random Loops
1Store animals and sounds in two matching lists
2Randomly pick an animal to quiz on each turn
3Check the answer and show ✓ or ✗ with score
⏱️
Project 03
🎮 Junior · Class 4–5
Times Table Trainer

Build a maths trainer that asks multiplication questions, tracks your score, and shows a "Well done!" message when you get 5 in a row correct.

Math Ops Counters Functions
1Generate random multiplication questions
2Keep a score counter and a streak counter
3Show celebration animation on 5-streak!
🌡️
Project 01
🔭 Explorer · Class 6
Weather Dashboard

Build a simple weather app that converts temperature between Celsius and Fahrenheit, shows an appropriate weather icon, and stores last 5 readings.

Functions Arrays DOM
1Write C↔F conversion functions
2Display matching weather icon (sunny/cloudy/rainy)
3Store last 5 inputs in an array and display them
📝
Project 02
🔭 Explorer · Class 7
To-Do List App

Create a fully working to-do list where you can add, tick off, and delete tasks. Tasks are sorted by priority and the list persists as a data structure.

Objects Sorting Events
1Store each task as an object {text, done, priority}
2Sort tasks by priority and render the list
3Add tick/delete buttons with event listeners
🎲
Project 03
🔭 Explorer · Class 8
Number Guessing Game

Build a smart number guessing game that gives "too high / too low" hints, tracks how many guesses you used, and shows a difficulty selector.

Logic Loops Recursion
1Generate a secret number within chosen difficulty range
2Give "Higher" / "Lower" hints after each guess
3Show guess count and optimal binary search solution
🔍
Project 01
⚡ Pro · Class 9–10
Search Algorithm Visualiser

Build an interactive tool that animates Linear Search vs Binary Search on an array. Show step count, time complexity, and let users compare side-by-side.

Algorithms Animation Big-O
1Implement both linear and binary search
2Animate each step with highlighted array cells
3Show step counter and O(n) vs O(log n) comparison
📊
Project 02
⚡ Pro · Class 10–11
Grade Calculator + Chart

Build a marks entry system that calculates GPA, plots a bar chart of subjects, identifies strongest and weakest subjects using array methods.

Data Structures Map/Filter Graphs
1Store subject marks as an array of objects
2Use map/filter/reduce to compute GPA and stats
3Render a bar chart using canvas or SVG
💬
Project 03
⚡ Pro · Class 11–12
Mini Chat App

Build a real-time style chat interface with message history, timestamps, user avatars, and a simple message queue data structure under the hood.

Queues OOP DOM Events
1Build a Message class with sender, text, timestamp
2Implement a queue to store message history
3Render chat bubbles and handle Enter-key submit
