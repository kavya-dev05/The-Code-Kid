## ✦ &nbsp; Getting Started

<br/>

**Prerequisites:** Node.js installed on your machine.

<br/>

```bash
# 1. Clone the repository
git clone https://github.com/kavya-dev05/The-Code-Kid.git

# 2. Navigate into the backend
cd The-Code-Kid/backend

# 3. Install dependencies
npm install

# 4. Set up your environment variables
cp .env.example .env
# → Add your Supabase credentials and Judge0 API key

# 5. Start the server
npm start

# 6. Open index.html with Live Server and you're live ✦
```

<br/>

### Getting a Judge0 API Key (for Pro Mode)

1. Visit [RapidAPI — Judge0 CE](https://rapidapi.com/judge0-official/api/judge0-ce)
2. Subscribe to the free tier and copy your API key
3. Add it to `backend/.env` as `JUDGE0_API_KEY=your_key_here`

## ✦ &nbsp; Environment Variables

Create a `.env` file inside `/backend` with the following:

```env
SUPABASE_URL=your_supabase_project_url
SUPABASE_ANON_KEY=your_supabase_anon_key
JUDGE0_API_KEY=your_judge0_api_key
PORT=3000
```

> ⚠️ Never push your `.env` file. It's already in `.gitignore`.
